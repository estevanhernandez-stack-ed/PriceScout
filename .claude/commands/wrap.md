# Wrap Session - Price Scout

Commit and push all changes in this repository.

## Step 1: Analyze Changes

```bash
git status
git diff --stat
git log --oneline -5
```

## Step 2: Generate Commit Message

Based on the changes, generate a commit message following conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `refactor:` Code refactoring
- `style:` Formatting changes
- `test:` Adding tests
- `chore:` Maintenance

## Step 3: Commit and Push

```bash
git add -A
git commit -m "your message here"
git push
```

## Step 4: Report Summary

After pushing, provide:
- Commit hash (short)
- Files changed count
- Brief description of what was committed
